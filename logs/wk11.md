# today's topic for loops and arrays
A for loop repeats a block of code a specific number of times.

you would use a for loop when:
• When you know how many times to repeat something
• Looping through arrays
• Repeating actions (like adding items to a page)

Initialization: Runs once before the loop starts; typically used to define a counter.
Condition: Evaluated before every iteration; if true, the loop continues. If false, it terminates.
Update Expression: Executed at the end of each iteration; usually used to increment or decrement the counter.

# arrays
An array stores multiple values in one variable.
• Each item has an index (position)
• Index starts at 0

A for loop is commonly used to go through every item in an array.

• i = 0 → start at first item
• i < colors.length → loop through all items
• i++ → move to next item each time
• colors[i] → access each item
