# js extras
- Local Storage (JavaScript) — Plain Notes

What it is

A way for websites to store data directly in the user’s browser
Part of the Web Storage API
Data is stored as key–value pairs

Use Cases

Saving user preferences (theme, language, settings)
Keeping users logged in (non-sensitive data)
Storing small amounts of app data for faster load times

Frontend (Client-Side)

• What the user sees and interacts with
• Runs in the browser (Chrome, Safari, etc.)
• Focuses on design, layout, and user experience
• Handles things like buttons, forms, animations

Backend (Server-Side)

• What happens behind the scenes
• Runs on a server
• Handles logic, data, and storage
• Communicates with databases
