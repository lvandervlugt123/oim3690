# iframe
An iframe (inline frame) embeds another webpage or external content inside your webpage. It’s often used to embed things like YouTube videos, Google Maps, or another website.

# canvas
The canvas element creates a space where you can draw graphics using JavaScript. It’s used for things like games, charts, animations, and drawings.

# audio and video controls

The audio element lets you play sound on a webpage, like music or podcasts.
The video element lets you embed and play videos directly on a webpage.

# session 15 intro to java script

- Thee beauty of Java script is that you can change any element on the page

- an array is a value that stores multiple values in a sequence


- Script Tag is how you include JavaScript in an HTML page. When placed at the bottom of , the browser has already read and built all the HTML elements before running the JS, so the code can safely find and interact with them.
- addEventListener: This tells the browser "watch this element, and when a specific thing happens, run this function." The "specific thing" is the event, a click, a form submission, a checkbox changing, a key being pressed, etc. The function that runs in response is called the event handler. You can attach as many listeners as you want to any element.
- Change in the dom: The DOM is the browser's live representation of your HTML — it's what's actually displayed on screen. When JavaScript changes the DOM, the page visually updates instantly without reloading. You can change text content, add or remove elements entirely, toggle CSS classes to change styling, or directly set style properties. Anything the user sees update on the page without a reload is JavaScript changing the DOM.
