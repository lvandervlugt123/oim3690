# Key differences between ID and Class

Class
• used for multiple elements
• written with . in CSS

ID
• used for one element
• written with # in CSS
• stronger in CSS specificity


# understanding forms
- forms allow the user to enter something and then which will trigger a /search, which then sends the data from the input to the endpoint
- the forms action is search (google babson example)
- the default method for a form action submit is get
- When you create a form in HTML, the action tells the browser where to send the search request. Many websites accept search queries through URL parameters.
    - For example:
    - Google uses q for the search query.
    - Amazon uses k for the search query.
